# 🤖 AI Resume Builder

A responsive AI Resume Builder built with HTML, CSS and JavaScript.

## Features
- Login Authentication
- Live Resume Preview
- Save Resume (Local Storage)
- Auto Load Saved Resume
- PDF Download
- Responsive Design

## Technologies
- HTML5
- CSS3
- JavaScript
- Local Storage
- jsPDF
